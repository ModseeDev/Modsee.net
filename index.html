Modsee

An open-source graphical modeling platform designed to simplify the development of finite element models for OpenSees and OpenSeesPy users.

Download Documentation

About Modsee

Modsee provides a modern, intuitive user interface for building, managing, visualizing, and exporting structural and geotechnical models, without the need for manual scripting.

Open Source

Freely available under GPLv3.

Cross-Platform

Available for Windows, macOS, and Linux.

Powerful Modeling

Supports core OpenSees element types with intuitive graphical interface.

Features

Designed to make OpenSees modeling more accessible

Graphical Model Builder

Create nodes, elements, materials, and boundary conditions visually without coding.

Element Support

Comprehensive support for 2D, 3D, springs, and soil element types.

Material Library

Comprehensive material library with management tools for custom materials.

Project System

Save and load your models with a robust project management system.

Export Options

Export models to both OpenSees TCL scripts and OpenSeesPy scripts.

Model Visualization

Visualize your results with a built-in viewer.

🛠️ Current Status

Modsee is in early development.

Initial architecture focuses on building a modular, extensible, and scalable foundation to support long-term growth.

Stay tuned for early alpha releases and development milestones!

Project Goals
  • Eliminate the complexity of manual OpenSees model scripting
  • Provide an accessible tool for students, researchers, and engineers
  • Support structural, geotechnical, and soil-structure interaction modeling
  • Foster a strong, community-driven open-source ecosystem

An open-source graphical modeling platform for OpenSees, built with passion for engineering modeling.
