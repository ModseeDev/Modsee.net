Modsee

An open-source graphical modeling platform designed to simplify the development of finite element models for OpenSees and OpenSeesPy users.

Get Started View on GitHub

Features

Designed with simplicity and power in mind

Graphical Model Builder

Intuitive interface for creating nodes, elements, materials, and boundary conditions.

Element Support

Comprehensive support for core OpenSees element types in 2D, 3D, springs, and soils.

Material Library

Extensive material library management for all your modeling needs.

Export Capabilities

Export models to both OpenSees TCL scripts and OpenSeesPy scripts.

Integrated Runner

Run your models directly with the integrated OpenSeesPy runner.

Visualization

Advanced model visualization with VTK or similar tools.

Getting Started

Follow these steps to get Modsee running on your system

Prerequisites

  • Python 3.7 or higher
  • pip (Python package installer)

Installation

1. Clone the repository:
git clone https://github.com/modsee/modsee.git
cd modsee
2. Install required dependencies:
pip install -r requirements.txt
3. For development, install development dependencies:
pip install -e ".[dev]"
4. To use OpenSeesPy integration:
pip install -e ".[openseespy]"

Running the Application

Using the runner script:
python run.py
Or using the console script:
pip install -e .
modsee

│   ├── models/             # Data models
│   │   ├── project.py      # Project data structure
│   │   ├── geometry/       # Geometry definitions
│   │   ├── materials/      # Material definitions
│   │   ├── elements/       # Element definitions
│   │   ├── mesh/           # Meshing algorithms
│   │   └── analysis/       # Analysis settings
│   ├── io/                 # Input/Output operations
